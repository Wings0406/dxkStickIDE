# 本地I2C地址映射关系
|拨码位|地址名称|I2C地址|
|---|---|---|
|0000 (0)|A|22 (0x16)|
|0001 (1)|B|23 (0x17)|
|0010 (2)|C|50 (0x32)|
|0011 (3)|D|51 (0x33)|
|0100 (4)|E|52 (0x34)|
|0101 (5)|F|53 (0x35)|
|0110 (6)|G|54 (0x36)|
|0111 (7)|H|55 (0x37)|
|1000 (8)|I|56 (0x38)|
|1001 (9)|J|57 (0x39)|
|1010 (10)|K|58 (0x3a)|
|1011 (11)|L|59 (0x3b)|
|1100 (12)|M|60 (0x3c)|
|1101 (13)|N|61 (0x3d)|
|1110 (14)|O|62 (0x3e)|
|1111 (15)|P|63 (0x3f)|

## 生成代码
```
mapper=lambda x:22+x+26*(x>1) # 地址与编号映射关系
line=lambda x:f"|{bin(x)[2:].rjust(4,'0')} ({x})|{chr(x+ord('A'))}|{mapper(x)} ({hex(mapper(x))})|" # 打印一行md

for i in range(16): # 输出结果
    print(line(i))
```
