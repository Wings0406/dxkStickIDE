# 本地I2C地址映射关系
|拨码位|地址名称|I2C地址|
|---|---|---|
|0000|A|22(0x16)|
|0001|B|23(0x17)|
|0010|C|50(0x32)|
|0011|D|51(0x33)|
|0100|E|52(0x34)|
|0101|F|53(0x35)|
|0110|G|54(0x36)|
|0111|H|55(0x37)|
|1000|I|56(0x38)|
|1001|J|57(0x39)|
|1010|K|58(0x3a)|
|1011|L|59(0x3b)|
|1100|M|60(0x3c)|
|1101|N|61(0x3d)|
|1110|O|62(0x3e)|
|1111|P|63(0x3f)|

## 生成代码
```
mapper=lmabda x:22+x+26*(x>1) # 地址与编号映射关系
line=lambda x:f"|{bin(x)[2:].rjust(4,'0')}|{chr(x+ord('A'))}|{mapper(x)}({hex(mapper(x))})|" # 打印一行md

for i in range(16): # 输出结果
    line(i)
```
